# cubic_4folds
Tables of cubic fourfolds and their geometric invariants, motivated by Manin's conjecture and the circle method.

The fewnomials data was generated by SageMath through CoCalc, using code written by Gemini 3 Pro.
For each non-conical, reduced, irreducible cubic fourfold X in P^5_Q defined by a sum of k pairwise distinct primitive monomials, the code computes the dimension and degree of the singular subscheme of X, as well as the degree of the dual variety of X, and whether the dual variety of X is a hypersurface or not.
